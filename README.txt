Steps to execute the project:
1. Run the program once with the "java Chat" command. Enter username when prompted, let's say Alice.
2. In another terminal, run the program again with the "java Chat" command. Enter username when prompted, let's say Bob.
3. In Alice's terminal, enter Bob's port number.
4. In Bob's terminal, enter Alice's port number.
5. Enter messages on both terminals to exchange chat messages.
6. Type transfer <filename> on either terminal to send a file to the other party.
